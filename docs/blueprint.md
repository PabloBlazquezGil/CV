# **App Name**: PabloBlazquezGil_CV

## Core Features:

- Interactive Sections: Display the different CV sections in a clear and navigable structure. Sections should include Experience, Education, and Skills.
- Skills Visualization: Visualize the user's skills in an engaging and interactive format. Explore AI to generate dynamic visual representations.
- Portfolio Showcase: Create a portfolio section where projects are displayed with expandable details.
- AI-Powered CV Tailor: Employ AI to analyze the user's CV and the description of a target job and give feedback on how to re-write the CV to more closely align it with the job. The AI should be given tools to help decide when the output document mentions various requirements of the job.
- Contact Options: Provide clear options to contact the user or download the CV directly.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect the natural inspiration.
- Background color: Light beige (#F5F5DC) for a soft, natural backdrop.
- Accent color: Warm brown (#A0522D) to highlight key elements.
- Body and headline font: 'Alegreya', a serif font for a balanced, modern feel. Note: currently only Google Fonts are supported.
- Use icons that are clean, modern, and relevant to each section of the CV.
- Maintain a clean, single-page layout to showcase professional details efficiently.
- Implement subtle transitions when navigating through different sections of the CV.
